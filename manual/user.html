予約方法

予約総合サイトで行事に申し込む方法を説明します。

← 使い方ガイドに戻る

予約の流れ

予約は、以下の流れで行います。

  1. 予約総合サイトを開く
  2. 参加したい行事を選ぶ
  3. 必要事項を入力する
  4. 入力内容を確認する
  5. 予約を送信する
  6. 予約完了画面を確認する
予約完了後は、画面に表示される内容を必ず確認してください。 必要に応じて、スクリーンショットを保存しておくと安心です。

1. 予約総合サイトを開く

LINEノートや案内文に掲載されている予約サイトのURLを開きます。

図1：予約サイトのトップ画面

トップ画面が表示されたら、予約したい行事を探してください。

2. 行事を選ぶ

表示されている行事一覧から、参加したい行事を選択します。

図2：行事選択画面
行事が表示されない場合は、受付期間外・対象者外・定員到達などの可能性があります。 詳しくは「困ったとき」のページを確認してください。

3. 必要事項を入力する

氏名、学年、連絡先、参加希望内容など、画面に表示されている項目を入力します。

入力時の注意

  1. 氏名はフルネームで入力してください。
  2. 学年・所属は正しく選択してください。
  3. 連絡先に誤りがないか確認してください。
  4. 備考欄がある場合は、必要な連絡事項を入力してください。

4. 内容を確認して送信する

入力内容に間違いがないか確認したうえで、送信ボタンを押します。

送信後に変更できない項目がある場合があります。 特に、参加日程・オプション・連絡先は送信前に必ず確認してください。

5. 予約完了を確認する

送信後、予約完了画面が表示されれば予約は完了です。

完了画面が表示されない場合や、エラーが出た場合は、重複して送信せず、まずは「困ったとき」のページを確認してください。

困ったときのページを見る

予約の変更・キャンセル

予約内容を変更したい場合や、参加できなくなった場合は、案内に記載されている方法に従ってください。

図3：予約変更・キャンセルに関する画面

変更・キャンセル時の注意

  1. 変更期限を過ぎている場合、サイト上で変更できないことがあります。
  2. キャンセル後に再予約する場合、定員の都合で再予約できない場合があります。
  3. 急な欠席や不明点がある場合は、担当者に連絡してください。
